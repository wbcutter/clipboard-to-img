## 🕹️ Cách sử dụng

### 📱 Trên điện thoại (iPhone / Android)
1. Mở web bằng Chrome / Safari
2. Bấm nút **"📋 Dán từ Clipboard"**
3. Trình duyệt hỏi quyền → bấm **"Cho phép"** (lần đầu)
4. Chọn ảnh từ **Thư viện / Chụp ảnh / Dán từ clipboard**
5. Xem preview → bấm **"⬇️ Xuất JPG"** → lưu ảnh

> ⚠️ Lưu ý: iOS Safari yêu cầu bấm nút "Dán" trong popup sau khi cho phép quyền.

### 🖥️ Trên máy tính (Windows / Mac)
1. Copy ảnh bất kỳ (`Ctrl+C` hoặc chuột phải → Sao chép hình ảnh)
2. Mở web → bấm nút **"📋 Dán từ Clipboard"**
3. Lần đầu trình duyệt hỏi quyền → bấm **"Cho phép"**
4. Ảnh hiện ra → bấm **"Xuất JPG"** → tải file
